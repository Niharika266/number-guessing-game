#Number Guessing Game

A simple Python mini project where the user tries to guess a randomly generated number.

#About the Project

This is a beginner-friendly Python project.
The program generates a random number between 1 and 100.
The user has to guess the correct number.
The game gives hints like "Too High" or "Too Low" until the correct number is guessed.

#Features

- Random number generation
- User input handling
- Hint system (Too High / Too Low)
- Runs until the correct number is guessed
- Beginner-friendly logic

#Technologies Used

- Python 3
- random module

#How to Run the Project

1. Install Python (if not installed)
2. Download or clone this repository
3. Open terminal or command prompt
4. Navigate to the project folder
5. Run the following command:

python miniproject_1.py

#Sample Output

Congratulations! You guessed it right.
